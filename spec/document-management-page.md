# 文档管理页面规格文档

## 1. 概述

将文档管理页面重构为两个独立的模块：
- **项目文档**：按项目维度管理文档
- **周报文档**：按周报维度管理文档

用户可以通过顶部Tab切换这两个模块，每个模块有独立的视图和功能。

## 2. 页面结构

```
┌─────────────────────────────────────────────────────────────┐
│  文档管理                                                    │
│  [项目文档] [周报文档]                                       │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  模块内容区域                                                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 3. 功能设计

### 3.1 项目文档模块

| 功能 | 说明 |
|------|------|
| 项目选择 | 下拉选择项目，支持"全部项目"选项 |
| 搜索 | 按文件名搜索 |
| 分类筛选 | 按文档分类筛选（7种分类） |
| 视图切换 | 网格视图 / 列表视图 |
| 文档卡片 | 显示缩略图、文件名、大小、分类、上传时间 |
| 操作 | 下载、预览、删除（需权限） |
| 统计 | 显示文档总数、存储占用 |

**分类配置**：
- screenshot: 工作截图
- design: 设计文档
- meeting: 会议记录
- progress: 进度报告
- code: 代码相关
- document: 一般文档
- other: 其他

### 3.2 周报文档模块

| 功能 | 说明 |
|------|------|
| 周报选择 | 选择周报查看关联文档 |
| 搜索 | 按文件名搜索 |
| 分类筛选 | 按文档分类筛选 |
| 视图切换 | 网格视图 / 列表视图 |
| 周报列表 | 按周报分组显示文档 |
| 文档卡片 | 显示缩略图、文件名、大小、分类、上传时间 |
| 操作 | 下载、预览、删除（需权限） |
| 统计 | 显示总文档数、存储占用 |

### 3.3 公共功能

| 功能 | 说明 |
|------|------|
| 上传文档 | 弹窗上传（项目文档/周报文档根据当前模块） |
| 预览 | 支持图片和PDF在线预览 |
| 下载 | 文档下载功能 |
| 删除 | 确认后删除（需权限） |
| 权限控制 | 根据用户角色控制操作权限 |

## 5. 技术实现

### 5.1 组件结构

```
src/views/DocumentManagement.vue     # 主页面（Tab切换）
src/components/document/
  ProjectDocuments.vue               # 项目文档模块
  ReportDocuments.vue                # 周报文档模块
  DocumentCard.vue                   # 文档卡片组件
  DocumentUploadModal.vue             # 上传弹窗（复用）
```

### 5.2 API接口

使用现有API：
- `GET /api/documents` - 获取所有文档
- `GET /api/documents/project/{projectId}` - 获取项目文档
- `GET /api/documents/report/{reportId}` - 获取周报文档
- `POST /api/documents` - 上传文档
- `DELETE /api/documents/{id}` - 删除文档
- `GET /api/documents/{id}/download` - 下载文档
- `GET /api/documents/{id}/preview` - 预览文档

### 5.3 数据过滤

**项目文档**：
- 过滤条件: `projectId` 字段
- 分类: `category` 字段

**周报文档**：
- 过滤条件: `reportId` 字段
- 分类: `category` 字段

## 6. 权限设计
